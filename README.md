# Online Food Ordering System
A C++ project demonstrating the use of Data Structures and OOP principles.

## Features:
- **Linked Lists:** Used to manage a dynamic list of orders.
- **Stack:** Implemented a custom Stack to handle "Undo" and "Redo" operations for orders.
- **Recursion:** Used to calculate optimal delivery time.
- **OOP:** Solid implementation of Classes (MenuItem, Customer, Order, System).
